# AI-GYM-REP-COUNTER

**This project is a real-time AI-powered Workout Tracker built using MediaPipe Pose estimation and Streamlit.**
  
-It uses your webcam to:
-Detect full-body pose landmarks
-Recognize different exercises (squats, curls, deadlifts, pull-ups & more)
-Count reps, sets, and total calories burned
-Provide real-time form feedback
-Auto-start rest timers between sets
-Log your entire workout into a CSV history file
